# 🚀 Easy-sh-helper

Минималистичный скрипт для быстрой установки AUR-хелперов (**yay** или **paru**) в Arch Linux.

### 🛠 Способ 1: Короткий путь (рекомендуемый)
Самый быстрый вариант запуска через сокращенную ссылку:
```bash
curl -sL is.gd/5fsboN | bash
```
### 🛠 Способ 2: Длинный путь (надежный)
```bash
bash <(curl -sL https://raw.githubusercontent.com)
```
